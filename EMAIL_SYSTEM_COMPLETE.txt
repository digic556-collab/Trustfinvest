🎯 HOW THE SYSTEM WORKS
═══════════════════════════════════════════════════════════════════════════════

Step-by-Step Flow:

1. USER ACTION OCCURS
   └─ Example: User deposits $5000

2. BACKEND RECEIVES REQUEST
   └─ POST /api/email/activity/log
   └─ Activity data included

3. ACTIVITY LOGGED TO FIRESTORE
   └─ Collection: user_activities
   └─ Record created with details

4. ACTIVITY LISTENER TRIGGERED
   └─ ActivityListener.autoSendEmail('deposit', data)
   └─ Routes to appropriate listener

5. USER PREFERENCES CHECKED
   └─ Query Firestore: newsletter_subscribers
   └─ Verify user enabled activity notifications

6. EMAIL SENT AUTOMATICALLY
   └─ emailService.sendDepositConfirmation()
   └─ Uses deposit_confirmation template
   └─ Includes app logo, user name, amount, reference

7. EMAIL LOGGED
   └─ Record created in email_logs collection
   └─ Timestamp, status, message ID recorded

8. USER RECEIVES EMAIL
   └─ Professional template
   └─ Personalized with user info
   └─ Action links to dashboard


📊 ACTIVITY TYPES & AUTO-EMAILS
═══════════════════════════════════════════════════════════════════════════════

1. NEW USER REGISTRATION
   Trigger:    User creates account
   Auto-Sent:  YES - Immediate
   Template:   newUserRegistration
   Content:    Welcome, getting started, dashboard link

2. WELCOME BACK (LOGIN)
   Trigger:    Returning user logs in
   Auto-Sent:  YES - On returning login (not first)
   Template:   welcomeBack
   Content:    Welcome back, what's new, portfolio link

3. DEPOSIT
   Trigger:    User deposits funds
   Auto-Sent:  YES - If activity emails enabled
   Template:   depositConfirmation
   Content:    Amount, reference, next steps, invest link

4. INVESTMENT
   Trigger:    User creates investment
   Auto-Sent:  YES - If investment emails enabled
   Template:   investmentCreated
   Content:    Plan, amount, ROI, earnings calculator

5. WITHDRAWAL INITIATED
   Trigger:    User requests withdrawal
   Auto-Sent:  YES - If activity emails enabled
   Template:   withdrawalInitiated
   Content:    Amount, method, timeline, status link

6. WITHDRAWAL APPROVED
   Trigger:    Admin approves withdrawal
   Auto-Sent:  YES - If activity emails enabled
   Template:   withdrawalApproved
   Content:    Approved notification, expected arrival

7. TASK COMPLETED
   Trigger:    User completes daily task
   Auto-Sent:  YES - If activity emails enabled
   Template:   taskCompleted
   Content:    Task name, reward earned, task link

8. DAILY EARNINGS
   Trigger:    Scheduled or on-demand
   Auto-Sent:  YES - If activity emails enabled
   Template:   earningsUpdate
   Content:    Daily earnings, total, investments, statistics

9. ACTIVITY ALERT
   Trigger:    Important event or unusual activity
   Auto-Sent:  YES - If alerts enabled
   Template:   activityAlert
   Content:    Activity type, details, timestamp, action needed

🔧 INTEGRATION POINTS
═══════════════════════════════════════════════════════════════════════════════

In Backend Routes:

1. User Registration Route:
   ```javascript
   await ActivityListener.onUserRegistration({
       email: user.email,
       firstName: user.firstName,
       userId: user.id
   });
   ```

2. Deposit Route:
   ```javascript
   await ActivityListener.onDeposit({
       userId: user.id,
       email: user.email,
       userName: user.firstName,
       amount: depositAmount,
       reference: transactionId
   });
   ```

3. Investment Route:
   ```javascript
   await ActivityListener.onInvestment({
       userId: user.id,
       email: user.email,
       userName: user.firstName,
       plan: investmentPlan,
       amount: investmentAmount,
       roi: planROI,
       duration: planDuration
   });
   ```

4. Withdrawal Route:
   ```javascript
   await ActivityListener.onWithdrawalInitiated({
       userId: user.id,
       email: user.email,
       userName: user.firstName,
       amount: withdrawalAmount,
       method: withdrawalMethod
   });
   ```

5. Task Route:
   ```javascript
   await ActivityListener.onTaskCompleted({
       userId: user.id,
       email: user.email,
       userName: user.firstName,
       taskTitle: task.title,
       reward: task.reward
   });
   ```


📲 FRONTEND INTEGRATION
═══════════════════════════════════════════════════════════════════════════════

Include API Client:
  <script src="/backend/client.js"></script>

Initialize:
  const api = new TrustFinvestAPI('http://localhost:5000');

Trigger Activities (Auto-Sends Emails):
  1. Register: api.registerUser(userData);
  2. Deposit: api.makeDeposit(depositData);
  3. Invest: api.createInvestment(investmentData);
  4. Withdraw: api.initiateWithdrawal(withdrawalData);
  5. Task: api.completeTask(taskId);

Update Preferences:
  api.updateEmailPreferences(email, {
      newsletter: false,
      activities: true,
      investments: true,
      tasks: false,
      alerts: true
  });
