💻 FRONTEND INTEGRATION:

1. Include the API client in your HTML:
   <script src="/backend/client.js"></script>

2. Initialize in JavaScript:
   const api = new TrustFinvestAPI('http://localhost:5000');

3. Use the API methods:
   
   Subscribe to newsletter:
   api.subscribeNewsletter('user@example.com', 'John', 'Doe');
   
   Log investment:
   api.logInvestment('user123', 'user@example.com', 'John', 5000, 'Premium', 12);
   
   Send notification:
   api.sendInvestmentNotification('user@example.com', 'John', 'user123', {
     plan: 'Premium',
     amount: 5000,
     roi: 12,
     duration: 30,
     maturityDate: '2024-07-15'
   });

See backend/integration-examples.js for 50+ code examples.

💡 TROUBLESHOOTING:

Server won't start?
  - Check port 5000 is not in use: lsof -i :5000
  - Try different port: PORT=5001 npm run dev
  - Check .env file exists

SMTP connection fails?
  - Verify credentials in .env
  - Check internet connection
  - Try: node test-smtp.js
  - Check firewall settings

Emails not sending?
  - Check email logs: GET /api/email/admin/logs/emails
  - Verify user preferences
  - Check spam folder
  - Review console logs

Database not connecting?
  - MongoDB is optional
  - Service works without it (API-only mode)
  - Check MONGODB_URI in .env if needed
